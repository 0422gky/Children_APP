# 页面跳转逻辑
## 1. 登录页面
- 登录成功 ：
  - 检查用户是否已选择兴趣
  - 未选择 → 跳转到 /interest-selection
  - 已选择 → 跳转到 /home
- 注册成功 ：
  - 跳转到 /interest-selection
## 2. 兴趣选择页面
- 完成选择 ：
  - 保存兴趣后跳转到 /home
## 3. 首页
- 顶部导航栏 ：
  - 个人图标 → 跳转到 /profile （当前用户）
- 好友列表 ：
  - 点击好友项 → 跳转到 /profile （其他用户，传递用户信息）
  - 聊天图标 → 跳转到 /chat （传递用户信息）
- 底部导航栏 ：
  - 首页 → 跳转到 /home
  - 活动 → 跳转到 /activity
  - 我的 → 跳转到 /profile
## 4. 活动页面
- 顶部导航栏 ：
  - 添加图标 → 跳转到 /create-activity
- 底部导航栏 ：
  - 首页 → 跳转到 /home
  - 活动 → 跳转到 /activity
  - 我的 → 跳转到 /profile
## 5. 个人资料页面
- 查看其他用户 ：
  - 返回按钮 → 返回到上一页
  - 发送消息 → 跳转到 /chat （传递用户信息）
- 查看当前用户 ：
  - 我的活动 → 跳转到 /activity
  - 我的好友 → 跳转到 /home
  - 底部导航栏：
    - 首页 → 跳转到 /home
    - 活动 → 跳转到 /activity
    - 我的 → 跳转到 /profile
